# 🧠 Quizzical

A responsive, accessible quiz app built with **React** and the [Open Trivia DB API](https://opentdb.com/). Users can customise their quiz by selecting the number of questions and difficulty level, answer multiple-choice questions, and receive instant scored feedback.

🔗 **Live Demo:** [milesquiz.netlify.app](https://milesquiz.netlify.app)

---

## Features

- Fetches randomised questions from the Open Trivia DB API on each game
- Client-side answer randomisation (correct answer is never always in the same position)
- Configurable number of questions and difficulty level
- Instant scoring and answer feedback on submission
- Accessible feedback using ARIA attributes
- Fully responsive layout
